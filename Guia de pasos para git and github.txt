Como conectar mis proyectos locales con github

Pasos

1. Crear repositorio en git hub
2. Crear init en carpeta del proyecto
3. Agregar el remoto del repositorio de github
4. Hacer git add .
5. Hacer git commit
6. Hacer git push origin master
